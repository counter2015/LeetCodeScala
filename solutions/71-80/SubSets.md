# Subsets

## Problem link
https://leetcode.com/problems/subsets/

## Description

Given a set of **distinct** integers, *nums*, 
return all possible subsets (the power set).

## Example

```
Input: nums = [1,2,3]
Output:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```
